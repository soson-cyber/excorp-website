## 01. What is EXLINK — EXLINK이란?

EXLINK은 실시간 XR 제작에 필요한 촬영·카메라 트래킹·가상 배경 렌더링·합성·미디어서버·송출 과정을 하나의 제어 흐름으로 통합한 EX 자체 개발 올인원 솔루션입니다. 여러 장비와 소프트웨어를 따로 연동하는 대신, 검증된 기술들을 EXLINK이 한 자리에서 조율합니다.

> 흩어진 실시간 XR 파이프라인을, 하나의 제어 흐름으로.

(이미지: EXLINK 노드 기반 실시간 XR 제어 흐름 — `/vp-workflow.png`)

## 02. Why Integrated — 왜 통합 솔루션인가

실시간 XR을 직접 구성하면 카메라·트래킹·렌더·미디어서버·송출이 제각각입니다. EXLINK은 이 과정을 하나의 흐름으로 묶어 복잡성을 줄이고, 더 적은 인력으로 더 빠르게 운영하게 합니다.

분산형 구성에서 EXLINK 통합으로의 대비:

| 분산형 구성 | → | EXLINK 통합 |
|-------------|---|-------------|
| 장비·소프트웨어가 제각각, 규격·제어 방식이 다름 | ↓ | 단일 제어 흐름에서 일관되게 운영 |
| 현장마다 연동·동기화 셋업에 시간 소요 | ↓ | 셋업·전환 시간 단축, 빠른 현장 대응 |
| 파트별 인력 분산으로 운영 부담이 큼 | ↓ | 운영자 1인 중심 워크플로우로 효율화 |

## 03. Architecture — 구성 / 아키텍처

CAPTURE → TRACKING → EXLINK CORE → RENDER → BROADCAST

파이프라인 노드(다이어그램):

| 노드 | 라벨 | 설명 |
|------|------|------|
| CAPTURE | 촬영 · 멀티캠 | 여러 카메라 영상을 실시간으로 입력받습니다. |
| TRACKING | 카메라 트래킹 | 카메라 위치·움직임을 정밀하게 추적합니다. |
| EXLINK (통합 코어) | 통합 코어 | 촬영·트래킹·렌더·송출을 한 흐름으로 조율합니다. |
| RENDER | 실시간 합성 | 가상 배경·그래픽을 실시간 렌더·합성합니다. |
| BROADCAST | 송출 · 녹화 | 실시간 송출과 동시 녹화를 진행합니다. |

## 04. Key Features — 핵심 기능

- **멀티캠·트래킹 자동 동기화** — 여러 카메라와 트래킹 데이터를 자동으로 맞춰 수동 보정 부담을 줄입니다.
- **가상 배경·그래픽 실시간 합성** — 가상 배경과 그래픽을 촬영 화면과 실시간으로 합성합니다.
- **라이브 송출·동시 녹화** — 실시간 송출과 녹화를 동시에 처리해 생방송과 아카이브를 함께 잡습니다.
- **운영자 1인 중심 제어 UI** — 핵심 제어를 한 화면에 모아 소수 인력으로 전체 흐름을 운영합니다.
- **실시간 처리** — 지연을 최소화한 실시간 처리(1프레임 미만 목표)로 라이브 환경에 대응합니다.
- **확장성** — 카메라 수·그래픽 소스·연결 기술을 현장 규모에 맞춰 확장합니다.

## 06. Connected Tech — 연결하는 기술

EXLINK은 검증된 전문 기술을 직접 만들지 않고, 한 흐름 안에서 조율합니다. 아래 파트너 제품은 EX가 국내 공급하는 기술이며, EXLINK이 이를 연결·통합합니다.

| 기술 | 역할 | 설명 |
|------|------|------|
| Aximmetry | 연결 | 실시간 3D / 버추얼 프로덕션 렌더 엔진을 EXLINK이 연결합니다. |
| Moverse AI | 연결 | 마커리스 AI 모션캡처를 EXLINK 흐름에 통합합니다. |
| RETracker | 연결 | 정밀 카메라 트래킹 기술을 EXLINK이 연동합니다. |
| 미디어서버 · 네트워크 | 연결 | 영상 소스 재생과 현장 네트워크를 제어 흐름 안에서 함께 운영합니다. |

※ Aximmetry · Moverse AI · RETracker는 각 제조사의 제품으로, EXLINK은 이를 연결·조율합니다. 자체 개발 영역은 통합 코어(EXLINK)입니다.
